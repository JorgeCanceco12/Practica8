Definición:

Las aplicaciones web progresivas son aplicaciones desarrolladas con tecnologías web como HTML, CSS y JavaScript, pero que incluyen características avanzadas para mejorar la experiencia del usuario, como la posibilidad de funcionar sin conexión, recibir notificaciones push y tener acceso a funciones nativas del dispositivo.
